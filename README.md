# TradeBotMini - OKX SOL 全仓合约交易机器人

一个针对 OKX 交易所 SOL-USDT 永续合约的自动化交易机器人，支持**斐波那契网格策略**、**限价单预挂**、动态止盈、SQLite 数据库存储和 Telegram 通知。

## 功能特点

- **斐波那契网格策略**: 根据斐波那契数列自动计算目标持仓，动态买卖
- **限价单预挂**: 在相邻斐波那契点位预挂买卖限价单，捕捉快速价格波动（wick）
- **随机价格偏移**: 使用 .2, .3, .6, .7 四个小数值避免被狙击
- **FIFO 记账**: 先进先出计算每笔真实盈亏
- **SQLite 数据库**: 持久化存储交易记录、持仓历史和统计数据
- **Telegram 通知**: 实时推送买入、卖出、盈亏通知
- **测试网支持**: 支持 OKX 模拟盘进行策略测试

## 斐波那契网格策略

斐波那契策略根据当前价格自动计算目标持仓，在关键斐波那契点位触发买卖。

### 限价单预挂功能（新）

机器人会在当前价格的相邻斐波那契点位自动预挂限价单：

1. **买入限价单**: 挂在下方斐波那契点位，价格略低于基准价
2. **卖出限价单**: 挂在上方斐波那契点位，价格略高于基准价
3. **随机偏移**: 使用 .2, .3, .6, .7 四个小数值，避免被狙击

#### 示例

当前价格 $135，持仓 15 张：
- 下方点位: Fib 0.550 @ $133.00，目标持仓 18 张
- 上方点位: Fib 0.618 @ $137.08，目标持仓 15 张

机器人自动挂单：
- **买入限价单**: $132.3 (或 $132.6, $132.7) 买入 3 张
- **卖出限价单**: $137.3 (或 $137.6, $137.7) 卖出 0 张（已达目标）

当买入单成交后：
1. 发送 Telegram 通知
2. 取消旧的卖出单
3. 在新的相邻点位重新挂单

### 交易逻辑

1. **启动时**: 根据当前价格计算目标持仓，市价买入到目标数量
2. **运行中**: 在相邻斐波那契点位预挂限价单
3. **订单成交**: 发送通知，更新数据库，重新挂单
4. **价格超出范围**: 取消所有挂单，停止交易

### 示例流程

```
启动，价格 $130 → 市价买入 20 张
预挂买入单 @ $126.7 (3张)
预挂卖出单 @ $133.3 (2张)

价格跌到 $126.7 → 买入单成交
  → 发送 Telegram 通知
  → 持仓变为 23 张
  → 重新挂单: 买入 @ $122.6, 卖出 @ $127.3

价格涨到 $137.3 → 卖出单成交
  → 发送 Telegram 通知（含利润）
  → 持仓变为 15 张
  → 重新挂单: 买入 @ $132.7, 卖出 @ $142.2
```

## 快速开始

### 1. 克隆仓库

```bash
git clone https://github.com/labubu-daydayone/tradeBotMini.git
cd tradeBotMini
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

### 3. 配置环境变量

```bash
cp .env.example .env
```

编辑 `.env` 文件：

```bash
# OKX API 配置
OKX_API_KEY=your_api_key_here
OKX_SECRET_KEY=your_secret_key_here
OKX_PASSPHRASE=your_passphrase_here
OKX_USE_TESTNET=true

# Telegram 配置
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here

# 斐波那契策略配置
FIBONACCI_PRICE_MIN=100.0
FIBONACCI_PRICE_MAX=160.0
FIBONACCI_MAX_POSITION=40

# 日志配置
LOG_LEVEL=INFO
CHECK_INTERVAL=5
```

### 4. 运行机器人

```bash
# 查看当前状态
python src/main.py --mode status

# 启动机器人
python src/main.py --mode run

# 使用测试网
python src/main.py --mode run --testnet

# 测试斐波那契点位和价格偏移
python src/main.py --mode test

# 手动买入
python src/main.py --mode buy --quantity 2

# 手动卖出
python src/main.py --mode sell --quantity 1
```

## 命令行参数

| 参数 | 说明 | 可选值 |
|-----|------|-------|
| `--mode` | 运行模式 | `run`, `status`, `buy`, `sell`, `test` |
| `--testnet` | 使用测试网 | 无需值 |
| `--quantity` | 买入/卖出张数 | 整数 |

## 数据库功能

交易机器人使用 SQLite 数据库持久化存储数据：

### 存储内容

- **交易记录**: 所有买入和卖出记录，包含价格、数量、盈亏等
- **持仓批次**: FIFO 记账的持仓批次
- **每日统计**: 每日交易次数、胜率、盈亏汇总

### 数据库文件

数据库文件默认保存在项目根目录：`trading.db`

### 查看数据

```bash
# 使用 sqlite3 命令行工具
sqlite3 trading.db

# 查看交易记录
SELECT * FROM trades ORDER BY created_at DESC LIMIT 10;

# 查看持仓批次
SELECT * FROM position_lots WHERE quantity > 0;

# 查看每日统计
SELECT * FROM daily_stats ORDER BY date DESC;
```

## 获取 API 配置

### OKX API

1. 登录 [OKX 官网](https://www.okx.com)
2. 进入 **账户** → **API**
3. 创建新的 API Key，勾选 **交易** 权限
4. 记录 API Key、Secret Key 和 Passphrase

### OKX 测试网（模拟盘）

设置 `OKX_USE_TESTNET=true` 即可使用模拟盘，无需真实资金。

### Telegram Bot

1. 在 Telegram 中搜索 [@BotFather](https://t.me/BotFather)
2. 发送 `/newbot` 创建机器人
3. 获取 Bot Token

### Telegram Chat ID

1. 搜索 [@userinfobot](https://t.me/userinfobot)
2. 发送任意消息获取 Chat ID
3. **重要**: 先向您的 Bot 发送一条消息，否则 Bot 无法向您发送通知
